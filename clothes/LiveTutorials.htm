BukaBaju , Template Keren untuk toko online Blogger

Posted by Gusti Adnyana Posted on 08.45 with No comments
BukaBaju Template,ya itulah nama template yang akan saya bagikan untuk anda yang ingin membuka lapak online alias berjualan dengan modal yang cekak. Template toko online blog ini Gratis dengan fitur desain yang simpel dan saya rasa cukup fleksibel untuk berjualan,kalau anda minat yang premium...!! Silahkan berkunjung ke www.dinotemplate.com .

Fitur-fitur di BukaBaju template :

  • Full Post Product.
  • Currency IDR ( Indonesia Rupiah )
  • Related Post.
  • Footer 4 Columns
  • Drop Down Menu.
  • Price Button.
  • Ads Ready header.
  • Widget ongkir.
  • widget bank.
Template ini sengaja tidak saya masukkan fitur shopping cart atau invoice email karna terkendala pada hosting yang sering mengalami gangguan. Tidak hanya itu,selain alasan di atas..!! Saya juga sudah pernah membagikan template yang memiliki fitur invoice MasgusMarket Template.

Gimana..?? Simpel bukan..?? saya membuat template ini dengan sedikit perpaduan CSS untuk mempercantik tampilannya. Masalah warna..!! perpaduan antara biru,merah dan hitam membuat semakin elegant tampilan template ini.Untuk panduannya,silahkan di simak tutorial berikut ini :

Persiapan sebelum menggunakan BukaBaju template :

  • Buatlah halaman cara pembelian produk di toko online anda.
  • Banner iklan ( jika ada )

Setting Posting Produk :

Template ini memiliki tampilan khusus dan tertata agar terlihat lebih keren.Maka dari itu anda harus menyesuaikan struktur posting yang satu dengan yang lainnya.
  • Buat postng baru.
  • Pastekan kode posting berikut pada mode HTML bukan Compose :
<div class="product_image">
<a class="cloud-zoom" href="http://4.bp.blogspot.com/-Qb0XOe0gSwM/UsDbutBkjBI/AAAAAAAABXo/mX8Ucixk_Bc/s1600/nike.jpg" rel="softFocus: true, position:'inside', smoothMove:2"><img border="0" class="item_thumb" src="http://4.bp.blogspot.com/-Qb0XOe0gSwM/UsDbutBkjBI/AAAAAAAABXo/mX8Ucixk_Bc/s1600/nike.jpg" /></a>
<span class="item_price">Rp 190.000</span></div>
<div class="product_describe">
Deskripsi Produk anda</div>

  • Ganti text yang mas gusti beri warna merah dengan data masing-masing produk seperti URL Gambar,Harga dan deskripsi produk
  • Klik publikasikan.

Setting HTML Template :

  • Klik menu Template.
  • pilih Edit HTML
  • Cari URL berikut ini,gunakan Ctrl+F untuk mempercepat pencarian :
<a class='belitdetail' href='http://demo-bukabaju.blogspot.com/p/cara-pembelian-barang.html'>Pesan Produk Sekarang</a>
  • Silahkan ganti url yang berwarna merah dengan URL halaman anda yang sudah dibuat.
  • Klik simpan Template.
